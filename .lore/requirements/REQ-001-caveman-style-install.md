# Caveman-style install experience

## Requirement

Lore Skill shall be installable as a global npm package using a command style similar to Caveman.

## Acceptance Criteria

- [ ] The package can expose a `lore-skill` CLI command.
- [ ] Users can install it globally with npm.
- [ ] Users can initialize agent instructions in a project directory.
- [ ] The Lore CLI itself remains a separate dependency.
